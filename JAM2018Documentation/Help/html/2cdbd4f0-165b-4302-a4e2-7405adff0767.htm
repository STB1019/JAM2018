GettingStarted

This section will aid you to get started on the project. First of all you need the following software installed on your system:

  • Unity: this is the main software we use to develop the game;

  • Visual Studio Community edition 2017 [OPTIONAL]: you can use Visual studio to code the C# scripts of Unity. It's optional because Unity come bundled with Monodevelop. However, Visual studio is recommended if you want to exploit every project defined in the repository (like the documentations ones); You can download the ide here.

  • Sand Castle [OPTIONAL]: if you want to build the documentation (the one you're reading right now) you need to install SandCastle in your system.

  • Unity Plugin for Visual Studio: you can code Unity C# scripts with Visual Studio thanks to this plugin. The page of this plugin is available here.

  • Graphviz: if you need to view a graphical representation of a graph inside the game, you can do so with IGraph methods. In order to use them you need to manually install GraphViz on your system. Graphviz is downloadable here. Furthermore, you need to add (after the installation) the "bin" folder inside Graphviz to the PATH: in this way the software can use the software dot.exe.

Installation of SandCastle

  1. Install SandCastle via the link here;

  2. If you want to setup a new documentation project, you can follow the directions in here;

Building of proejct

  1. In order to play the game, you need to compile the support library called "SharpLibrary". To do so, open visual studio and open the relative solution;

  2. Select the whole solution inside "solution explorer", right clck on the solution item and choose "build solution";
