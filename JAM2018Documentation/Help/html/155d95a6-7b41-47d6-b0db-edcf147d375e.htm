Getting Started

First of all you need the following software installed on your system:

  • Unity: this is the main software we use to develop the game

  • Visual Studio Community edition 2017 [OPTIONAL]: you can use Visual studio to code the C# scripts of Unity. It's optional because Unity come bundled with Monodevelop. However, Visual studio is recommended if you want to exploit every project defined in the repository (like the documentations ones); You can download the ide herehttps://www.visualstudio.com/downloads/.

  • Sand Castle [OPTIONAL]: if you want to build the documentation (the one you're reading right now) you need to install SandCastle in your system.

  • Unity Plugin for Visual Studio: you can code Unity C# scripts with Visual Studio thanks to this plugin. The page of this plugin is available herehttps://www.visualstudio.com/vs/unity-tools/.

  • Graphviz: if you need to view a graphical representation of a graph inside the game, you can do so with IGraph methods. In order to use them you need to manually install GraphViz on your system. Graphviz is downloadable herehttp://www.graphviz.org/Download..php. Furthermore, you need to add (after the installation) the "bin" folder inside Graphviz to the PATH: in this way the software can use the software dot.exe.
