NotificationCenter Class

Represents an object that accepts subscriptions and dispatches notifications to other objects. See DefaultNotificationCenter for an example on how to use a notification center.

SendASynchronousNotification
Sends a notification to every suscribed objects asynchronously. This means that the notification center will not wait until the callback for the subscriber "i" fully completes before starting the execution of the callback of the subscriber "i+1". Instead, all callbacks will start at the same time.
SendSynchronousNotification
Sends a notification to every suscribed objects synchronously. This means that the notification center will wait until the callback for the subscriber "i" fully completes before starting the execution of the callback of the subscriber "i+1".
